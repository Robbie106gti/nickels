CC-CHGM - Continuous Horizontal Grain Match »
Code information

Code: CC-CHGM
Specify: -
Description:
  • Continuous Horizontal Grain Match.
  • To have continuous matching grain across multiple cabinets/items, attach CC-CHGM to each item except the last. On the last item attach a note stating items to Horizontally Grain Matched from left to right - see Elevation. See Screenshot
Old code(s): -

By default, grain matching will only be on the individual cabinet and span only the width of that cabinet.

HGM vs CHGM

By default, a door classified as HG (Horizontal grain) indicates that doors and/or drawers on an individual cabinet will be cut from the same piece of material resulting in a horizontal grain match spanning the width of the individual cabinet.

Effective immediately all HG doors, such as Alpha HG will now be referred to as HGM. The grain will still be matched only on the individual cabinet.

HGM

Example of HGM applied to each cabinet but not continuously across multiple cabinets/items.

In order to have grain matching span a run of cabinetry and parts each item will require an attached
CC-CHGM (Continuous Horizontal Grain Match). If these charges are not applied, Nickels will only match the grain spanning the width of the individual cabinet.


CHGM

To have continuous matching grain across multiple cabinets/items, attach CC-CHGM to each item except the last. On the last item attach a note stating items to Horizontally Grain Matched from left to right - see Elevation. See Screenshot
