Customization Charges


  • Customization charges are applied to cabinets and accessories or as stand-alone items. For further information, contact Order Desk.

Cabinet modifications

CC-FCBNC - Finish cabinet bottom no charge »
Code information

Code: CC-FCBNC
Specify: -
Description:
  • Finish cabinet bottom no charge.
  • Nickels will apply when necessary.
Old code(s): -

Typically, is used when the exterior material and interior material drop down menu are matching and a FCB is required.
CC-PLYTOP - Plywood top set shelf »
Code information

Code: CC-PLYTOP
Specify: Size
Description:
  • Plywood top set shelf.
  • Can also be used to replace strechers.
  • To strenghten the cabinet to support the extra weight of the appliances, Farmsinks & Rangtops.
Old code(s): CC-D5

Open cabinets and cabinets available over 39” wide will automatically be provided with a plywood top set shelf.

Drawer modifications

CC-DCR-D - Decrease depth of standard Drawer Box or Rollout/Slide - specify size »
Code information

Minimum depth: 10"
Description:
  • Decrease depth of standard Drawer Box or Rollout/Slide.

CC-BOXONLY - Drawer box only (delete slides) »
Code information

Code: CC-BOXONLY
Specify: -
Description:
  • Drawer box only.
Old code(s): CC-R1

This deletes the slides from the drawer box.
CC-INCR-D - Increase depth of a Drawer Box or Rollout - specify size »
Code information

Code: CC-INCR-D
Specify: Size (max. 29 3/8") or slide lenght
Description:
  • Increase depth of a Drawer Box or Rollout.
Old code(s): CC-A8, CC-B5

Drawers scale automatically up to 24”. Cabinets over this size will need this code attached to increase the depth of the drawer box.
