Drawer modifications

CC-DRM-H - Modify height of standard Drawer Box or Rollout/Slide - specify size »
Code information

Code: CC-DRM-H
Specify: Height of drawer box side
Resctrictions height: min. 3" & max. 14"(depending on draw species)
Description:
  • Modify height of standard Drawer Box or Rollout/Slide.
Old code(s): CC-A2H, CC-A9

Could be used when needing to make space for Gas or Electric lines.
When adding in a drawer in a base cabinet this code can be used to optimally scale the drawer box in the cabinet. For use as recycling or other bins.

CC-DRM-D - Modify depth of standard Drawer Box or Rollout/Slide - specify size »
Code information

Code: CC-DCR-D
Specify: Depth of drawer box side
Restrictions depth: Min. 10" & Max. 29 3/8" (or slide length, depending on draw species)
Description:
  • Change depth of standard Drawer Box or Rollout/Slide.
Old code(s): CC-A2L, CC-A8, CC-B5

Could be used when needing to make space for Gas or Electric lines.
Drawers scale automatically up to 24”. Cabinets over this size will need this code attached to increase the depth of the drawer box.

CC-FJ - Factory joint on veneer finish panel over 48" wide - per joint - specify seam placement »
Code information

Code: CC-FJ
Specify: seam placement
Description:
  • Factory joint on finish panel or counter over 48" wide - per joint
  • Panel or counter material is ordered as a square footage item specified as width x height.
  • Grain (if applicable) will run vertically, or in the direction of the selected exterior material if there is a choice of VG or HG.
  • Panels are joined together by 3" domino joints.
Old code(s): CC-D3

Two nailing strips are applied to the panel (as seen in back view) for transport only and are removed before installation.
Could also be used to extend a painted panel over 120".
